Scrapshare

Scrapshare is a cross-platform application that allows users to post ads for scrap raw material for other people to pick up for free. This project was developed by Hazem and I as an attempt to promote reusibility and incrase the life cycle of raw material; thus, decreasing CRD waste. We developed the app using flutter in 24 hours and pitched the idea at the Mount Pearl Hackathon and won second place :)

Skittles Sorter

Overview

The Skittles Separator illustrates some of the fundamentals of C++ implemented into a microcontroller, the Arduino Uno. This project consists of a mechanism that is able to separate Skittles candy based on color into designated compartments. The project is also programmed to keep track of how many skittles of each color were passed through a color sensor. The user places a random sample of skittles into a funnel on the top of the machine and then the machine is initiated via turning a rotary dial and the separation process begins.

Purpose

My partner, Hazem Ennacuer and I really wanted to only eat the red and purple skittles ... so the Skittles Seperator was born
